# 🌸 KNN Classification using Iris Dataset

A Machine Learning classification project that predicts the species of an Iris flower using the **K-Nearest Neighbors (KNN)** algorithm. This project demonstrates the complete machine learning workflow, from data preprocessing to model evaluation and visualization.

---

## 📌 Project Overview

The Iris dataset is one of the most popular datasets for learning machine learning classification algorithms. In this project, the KNN algorithm is trained to classify Iris flowers into one of three species:

* 🌼 Iris Setosa
* 🌿 Iris Versicolor
* 🌺 Iris Virginica

The notebook includes detailed explanations for each step, making it suitable for beginners learning supervised machine learning.

---

## 🎯 Objectives

* Understand the K-Nearest Neighbors (KNN) algorithm.
* Load and explore the Iris dataset.
* Preprocess the data for model training.
* Train a KNN classifier.
* Evaluate model performance using different metrics.
* Visualize the prediction results.

---

## 📂 Dataset

This project uses the **Iris Dataset** provided by Scikit-learn.

**Features:**

* Sepal Length
* Sepal Width
* Petal Length
* Petal Width

**Target Classes:**

* Setosa
* Versicolor
* Virginica

Dataset Size:

* **150 Samples**
* **4 Features**
* **3 Classes**

## 📚 Machine Learning Algorithm

### K-Nearest Neighbors (KNN)

KNN is a supervised learning algorithm that classifies a new data point based on the majority vote of its **K nearest neighbors** using a distance metric such as Euclidean Distance.
